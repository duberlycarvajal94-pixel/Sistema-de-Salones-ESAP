⚙️ Configurar Firebase (sincronización en tiempo real)

Firebase es el servicio gratuito de Google que permite que todos vean las reservas en tiempo real desde cualquier dispositivo.
  1. Vaya a console.firebase.google.com e inicie sesión con su cuenta Google.
  2. Crear proyecto → nombre: salones-esap → desactive Analytics → Crear.
  3. Menú izquierdo: Compilación → Realtime Database → Crear base de datos → Modo de prueba → Habilitar.
  4. Copie la URL que aparece:
    https://salones-esap-xxxxx-default-rtdb.firebaseio.com
  5. Abra este archivo HTML con el Bloc de notas, busque la línea:
    const FIREBASE_URL = ''; y pegue la URL entre las comillas. Guarde.
  6. Comparta el archivo HTML por correo, WhatsApp o súbalo a OneDrive.
